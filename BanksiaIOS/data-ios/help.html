Make move: Use one of two ways:

  • drag and drop: drag the piece you want and drop it to the new position
  • tap-tap: first tap will mark the piece you want to move, second tap is the new one

Castling moves: Move the King to his target squares or to drag the King and drop to the Rook (King captures his own Rooks)

Enlarge board: Long tap (press and hole about 1 second) to enlarge the board. Long tap again to reduce the board size into normal one. When the board is enlarged, you may use a finger to move board around to see hidden parts.

Menu Game

  • New game: create a new game
  • Paste: paste a new game in FEN or PGN
  • Copy/Email: can copy/email current position in FEN or image, or whole game in PGN format
  • Flip: flip the board
  • Edit position: Edit the current position

Edit position

  • Move or add pieces: drag pieces inside the board or in bottom bar and drop into new positions
  • Remove pieces: drag pieces inside and drop outside the board
  • Clear button: clear the board to empty
  • Startpos button: set the board to start position
  • Cancel button: discard all changed and stop editing
  • Done button: accept the edited position and stop editing

Level

  • Analyse on fly: if it is on, the engine will continue computing and you are freely to visit any move (go first, prev, next, last move) or make new moves. Whenever the current displaying position changed, the engine will be updated immediately and starts computing with new positions. You can see it's computing info via analysis displaying, analysis arrows. When the function is on, all items in right below section will be disabled (can't select)
  • Human white, computer black: you play white, chess engine plays black side
  • Human black, computer white: you play black, chess engine plays white side
  • Human both: you play both sides
  • Timer: you can choose one from different timers:
  • Standard: time-move-increment. If move number is zero, it becomes Fischer timer
  • Depth: set maximum depth for engine.
  • Move time: set max time for one side to make move
  • Nodes: set limits nodes for engines

    • Note: For depth and node modes: Human (you) won't have any limits.

    Strength

  • Threads: set maximum number of threads the engine can use for computing. Typically it should be set below the number of total physical cores
  • Skill levels (for Stockfish only): select from 20 (max, strongest) to 0
  • NNUE mode (for Stockfish only): select between modes of using NNUE
  • Use opening book: if it is on, the app will use moves from a small book as many as possible. If it's out of the book it will let the engine starts computing
  • Benchmark (Stockfish only): let Stockfish to calculate benchmark

Menu Move

  • Go first: go to the first move
  • Go previous: go to the previous move
  • Go next: go to the next move
  • Go last: go to the last move
  • Takeback: takeback the last move

Options

  • Show coordinate: on/off displaying board coordination
  • Show legal moves: if it is on, whenever you start dragging or select a piece, the app will display all legal positions you can move that piece to
  • Show analysis: if it is on, when an engine starts computing, the app will display all computing information such as speed (NPS), score, node, pv...
  • Show analysis arrow: show arrow on the move the engine is considering to make
  • Show analysis on move list: if is on, the app will collect and display computing information from the engine and display them on the move list as comments. The format is {node/speed time}
  • Sound: if it's not off, the app will make sound whenever a move is made. The sounds could be simple ones or human void
  • Piece set: select piece style from 5
  • Board colour: select board colour from 3
