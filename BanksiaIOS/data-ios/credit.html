BanksiaGUI for iOS is an open source, developed based on Banksia GUI (for desktop computers, Windowns/Linux/macOS) and open source Banksia.

Please help to improve the app and/or donate, thanks for any help.

As a user:

  • Download and use
  • Allow the app displaying ads
  • Report bugs, make suggestions
  • Give the app 5 stars and/or good comments
  • Donate

As a programmer:

  • Improve the app
  • Add more engines

BanksiaGUI is in active development, new functions, bug fixes and new versions will be frequently added. Any helps, suggestions, bug reports are more than welcome.
